Detailed Description

An activity representing a list of Items. This activity has different presentations for handset and tablet-size devices. On handsets, the activity presents a list of items, which when touched, lead to a ItemDetailActivity representing item details. On tablets, the activity presents the list of items and item details side-by-side using two vertical panes.
